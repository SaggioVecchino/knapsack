Instances of 0/1 Knapsack Problem

0/1 Knapsack problem files

File name: kp_n_wmax

kp: name of the instance of 0/1 knapsack problem

n: number of items

wmax: knapsack capacity

Example

n wmax

v1 w1

v2 w2

: :

vi wi

: :

vn wn

vi: profit of item i

wi: weight of item i
